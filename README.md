# 1. 项目概述
本项目是基于 Qt 框架实现的 C/S 架构 ATM 模拟系统，包含：

1.Qt 客户端：模拟 ATM 物理终端界面

2.服务器端：处理业务逻辑和数据存储

3.通信协议：基于 RFC-20232023 协议实现数据交互

4.数据库：MySQL 存储用户账户信息和交易记录

# 2.功能模块
## 2.1 客户端功能
用户注册、用户登录、密码验证、余额查询、取款功能、存款功能。
##  2.2 服务端功能
账户添加、账户验证、余额查询、余额修改、互动日志。
